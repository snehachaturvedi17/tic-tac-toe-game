# Tic Tac Toe (X-O Game) 🎮

## 📖 Overview
A simple interactive Tic Tac Toe (X-O) game built using JavaScript.  
This project demonstrates DOM manipulation, game logic handling, and event-driven programming.

## 🚀 Features
- Two-player mode
- Win detection logic
- Game reset functionality
- Clean UI

## 🛠 Tech Stack
- HTML
- CSS
- JavaScript (ES6)

## ▶️ How to Run
1. Clone the repository
2. Open `index.html` in your browser
